# BIOT74000 - Bioinformatics E-Portfolio Template

Welcome! This repository is the starting point for your **Final Project Part B: E-Portfolio**.

**Do not start editing files randomly.** Follow the step-by-step guide posted on
eConestoga: *"BIOT74000 - Building Your E-Portfolio on GitHub Pages."* It walks
you through everything in order, assuming no prior GitHub experience.

---

## The four sections you will complete

Your portfolio has four required sections. Each one is a single file in the
`_pages` folder that you edit in your web browser:

| Section | File to edit | Points |
|---|---|---|
| About | `_pages/about.md` | 10 |
| Bioinformatics Toolkit | `_pages/toolkit.md` | 30 |
| Skills Snapshot | `_pages/skills.md` | 30 |
| Capstone Connection | `_pages/capstone.md` | 20 |

Professional presentation and effective use of the tools is worth the
remaining **10 points**.

---

## The three things you must set up first

Before editing any section, open **`_config.yml`** and edit the top section as
described in the guide. You must set:

1. `url` - your GitHub Pages address
2. `baseurl` - must match your repository name exactly
3. `repository` - your username and repository name

Getting `baseurl` wrong is the most common reason a site's styling breaks.
The guide explains exactly what to enter.

---

## Where things live

- `_config.yml` - site-wide settings (edit the top section only)
- `_pages/` - your four required portfolio sections
- `_data/navigation.yml` - the top menu (you do not need to change this)
- `images/` - upload profile photos or figures here
- `files/` - upload PDFs or other documents here
